Visualization of your Data made Easy !

Load Your Data

Browse any csv/excel/tsv file. Upload the file to the VizAt tool. The dimentation of the file is not a concern for the tool. It selects any file that you need to upload and Vizualize. The dimentation or the attributes that requires are been selected by the user at the next step.

Choose the attributes

After selecting the data files, the content is displayed to the user. The user can then choose the attributes he needs to compare. The basic preprocessing steps can also be applied on this stage, where the data is prepared for generating Vizualizations.

Vizualize the Data

Vizualization of data is made possible with the help of JavaScript libraries. Where the users can select the type of graph he needs, the application makes them possibile to the user, the user can save the graph. Custom changes they can make by tweeking the parameters available.
